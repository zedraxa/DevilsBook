跨平台暗色笔记应用——恶魔之书。

Devil's Book 是一款高品质的暗色系数字手稿应用，专为触控笔优先的书写体验而打造。以沉浸式的电影感界面，为您提供卓越的手写笔记体验。

它的设计在尽可能简洁和直观的同时，仍然为您提供实际使用的独特功能。Devil's Book 可以在您所有的设备上使用，无论大屏幕还是小屏幕，并在它们之间无缝同步。

值得注意的是，当您使用暗色模式时，它可以反转笔记颜色。这样您就可以在黑色背景上用白色墨水书写，在光线较暗的环境下看得更加清楚。图像和 PDF 文件也可以被反转，因此您仍然可以使用数字打印输出或教科书。

Devil's Book 使用双密码系统来保护您的笔记，除了您之外，任何人都无法查看您的笔记，即使他人完全控制了服务器。

这款应用程序完全开源，任何人都可以查看源代码，了解它的具体操作和数据处理方式。

Devil's Book 的荧光笔利用画布合成来渲染，其处理重叠和保持颜色一致性的方式与传统纸张一致甚至更优。

Devil's Book 可以满足您整理笔记的一切需求。您可以随心所欲地创建文件夹内的文件夹，嵌套文件夹的数量不受限制。即使笔记被深埋在嵌套文件夹中，您仍然可以通过主屏幕上的最新笔记轻松访问它。

使用 Devil's Book，探索一种全新的方式来捕捉和组织您的想法。无论您是学生、专业人士还是创意人士，Devil's Book 都是您值得信赖的数字手写工具。
