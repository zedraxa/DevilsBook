跨平台暗色筆記應用——惡魔之書。

Devil's Book 是一款高品質的暗色系數位手稿應用，專為觸控筆優先的書寫體驗而打造。以沉浸式的電影感介面，為您提供卓越的手寫筆記體驗。

它的設計在盡可能簡潔和直觀的同時，仍然為您提供實際使用的獨特功能。Devil's Book 可以在您所有的裝置上使用，無論大螢幕還是小螢幕，並在它們之間無縫同步。

值得注意的是，當您使用深色模式時，它可以反轉筆記顏色。這樣您就可以在黑色背景上用白色墨水書寫，在光線較暗的環境下看得更加清楚。圖片和 PDF 檔案也可以被反轉，因此您仍然可以使用數位教科書而無需煩惱。

Devil's Book 使用雙密碼系統來保護您的筆記，除了您之外，任何人都無法查看您的筆記，即使他人完全控制了伺服器。

本應用程式完全開源，任何人都可以查看原始碼，了解它的具體運作方式以及如何處理您的資料。

Devil's Book 的螢光筆利用畫布合成來渲染，其處理重疊和保持顏色一致性的方式與傳統紙張一致甚至更優。

Devil's Book 具有使筆記井井有條所需的一切功能。在無限巢狀的資料夾中建立資料夾，盡情發揮您的創意，而且對於巢狀資料夾的數量沒有限制。即使筆記可能深藏在巢狀的資料夾中，您仍然可以輕鬆存取，因為最近的筆記始終顯示在主螢幕上。

透過 Devil's Book，發現一種全新的捕捉和組織思緒的方式。無論您是學生、專業人士還是具有創造力的思維者，Devil's Book 都是您在數位手寫中信賴的夥伴。
